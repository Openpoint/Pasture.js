Pasture.js
Pasture is a framework for distributed, browser based computing systems.

It is has a modular structure built on top of Node.js and Socket.io. These establish web sockets between HTML5 browser clients and your node server. Pasture provides a common language to pass instructions bi-directionally between your server, the browser JS Window and browser Web Workers.
